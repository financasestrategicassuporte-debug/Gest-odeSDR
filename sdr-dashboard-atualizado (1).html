Webhook RD Station
COMO INTEGRAR

Quando uma oportunidade mudar de etapa no RD Station, o sistema dispara automaticamente o fluxo configurado para aquela etapa.

Configure a URL do servidor nas ⚙ configurações
1. RD Station CRM → Configurações
2. Integrações → Webhooks → Novo Webhook
3. Cole a URL acima
4. Evento: "Oportunidade mudou de etapa"
5. Também ative: "Mensagem recebida no chat"
6. Salvar e testar
